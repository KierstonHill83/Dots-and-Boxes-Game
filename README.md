# The Box Game

### Purpose

This is a game where you will complete borders around boxes. The last person to complete the box gets a point. The person with the most points at the end of the game wins.

### Feature List

1. Box Grid
2. Player1 Scoring
3. Player2 Scoring
4. Alert When There is a Winner
5. Reset Button

### Technology

* CSS
* Javascript
* jQuery
* Jasmine
